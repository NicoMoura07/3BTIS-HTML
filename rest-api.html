O que é REST API?

REST é um modelo usado para criar sistemas que trocam dados pela web. Ele funciona usando métodos HTTP e geralmente retorna informações em JSON.

  • GET: buscar dados
  • POST: enviar dados
  • PUT: atualizar
  • DELETE: remover

Como funciona?

Uma API oferece URLs (endpoints) para o cliente acessar. Exemplo:

GET /api/usuarios

Resposta típica em JSON:

{
        "id": 1,
        "nome": "Ozzy"
        }

Java (Back-end):

Com Spring Boot é possível criar APIs de forma simples.

@RestController
        public class ApiController {

            @GetMapping("/api/msg")
            public String msg() {
                return "Exemplo de API em Java.";
            }
        }

Exemplo retornando JSON:

public record User(String nome,int idade){}

        @GetMapping("/api/user")
        public User user(){
            return new User("Ozzy",15);
        }

JavaScript (Front-end):

Para consumir uma API no navegador:

fetch("/api/user")
            .then(r => r.json())
            .then(d => console.log(d.nome));

Exemplo Prático (piadas.html):

Foi feita uma página com um botão que buscava piadas de uma API pública. Ao clicar, o JavaScript usava fetch() para pegar o texto e mostrar na tela. 

fetch("https://v2.jokeapi.dev/joke/Any?lang=pt")
        .then(r => r.json())
        .then(data => {
            let texto = data.type === "single"
                ? data.joke
                : `${data.setup} ... ${data.delivery}`;

            document.getElementById("piada").innerText = texto;
        });
